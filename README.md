# DEAD CITY — PURGE 🧟

Um shooter de zumbis em primeira pessoa que roda direto no navegador (PC e celular). Arquivo único, sem instalação.

## 🎮 Jogar agora

Depois de publicado no GitHub Pages, o link fica assim:

```
https://SEU-USUARIO.github.io/dead-city/
```

Abra no navegador (Chrome, Safari, Edge, Firefox) — funciona no celular e no computador.

## 🕹️ Controles

**PC**
- `WASD` — mover
- `MOUSE` — olhar (360° + cima/baixo)
- `CLIQUE` — atirar
- `R` — recarregar
- `1` / `2` / `3` — trocar de arma
- `SHIFT` — correr
- `🔊` (canto direito) — mudo

**Celular**
- Joystick (esquerda) — mover
- Arrastar na tela (direita) — olhar
- Botão FOGO — atirar
- Botão REC — recarregar

## 🧟 O jogo

- 5 tipos de zumbi (Normal, Rápido, Tank, Seleção #10 e o Gordo Cuspidor que atira vômito verde)
- 3 armas espalhadas pelo mapa: Pistola, Metralhadora e Lança-Foguete
- Sistema de ondas (waves) com dificuldade crescente
- Colete (armor), kits de vida e munição
- Som e música sintetizados + fanfarra de abertura original

## 🚀 Como publicar (GitHub Pages)

1. Crie um repositório novo no GitHub chamado, por exemplo, `dead-city`.
2. Suba o arquivo `index.html` (e este README) para o repositório.
3. No repositório, vá em **Settings → Pages**.
4. Em **Build and deployment → Source**, escolha **Deploy from a branch**.
5. Em **Branch**, selecione `main` e a pasta `/ (root)`. Clique em **Save**.
6. Aguarde ~1 minuto. O link público vai aparecer no topo dessa mesma página.
7. Compartilhe o link com seus amigos. Pronto! 🎉

## 📝 Notas

- Tudo (gráficos, sons, música) é gerado/embutido no próprio arquivo `index.html`. Não há dependências externas.
- O áudio só inicia após o primeiro toque/clique (regra dos navegadores).
